Tipos de Algoritmos

Algoritmos de Força Bruta

Definição

Os algoritmos de força bruta tentam resolver problemas experimentando todas as soluções possíveis. São simples de implementar, mas geralmente ineficientes para problemas grandes.

Exemplo

Pesquisa linear: o algoritmo verifica cada elemento de uma lista até encontrar o que procura.

Algoritmos de Recursão

Definição

A recursão envolve uma função que se chama a si própria para resolver subproblemas menores do problema original.

Exemplo

Fatorial de um número: O cálculo de n! pode ser feito chamando repetidamente a função fatorial até atingir 1.

Algoritmos de Ordenação

Definição

Os algoritmos de ordenação organizam os dados (listas ou arrays) em ordem crescente ou decrescente. Existem vários algoritmos com diferentes eficiências.

Exemplos

Bubble Sort

Ordenação por comparação repetitiva de elementos adjacentes.

Quick Sort

Ordenação através da divisão recursiva da lista com base num pivô.

Merge Sort

Ordenação recursiva que divide a lista em partes menores e as ordena ao juntá-las.

Algoritmos de Busca

Definição

Os algoritmos de busca encontram um elemento numa estrutura de dados. Podem ser lineares ou binários, dependendo da organização dos dados.

Exemplos

Busca Linear

Verifica cada elemento, um a um, até encontrar o desejado.

Busca Binária

Efetua a busca dividindo a lista ao meio repetidamente, mas requer que a lista esteja ordenada.

Algoritmos Gulosos

Definição

Os algoritmos gulosos tomam a decisão mais vantajosa localmente, na esperança de que a solução global seja ótima.

Exemplo

Problema da mochila fracionária: o algoritmo escolhe os itens de maior valor por peso até encher a mochila.

Algoritmos de Retrocesso

Definição

Os algoritmos de retrocesso constroem soluções passo a passo, voltando atrás quando se encontra um caminho sem saída. São muito usados em problemas de otimização e quebra-cabeças.

Exemplo

Problema das n-rainhas: posicionar n rainhas num tabuleiro de xadrez sem que elas se ataquem.
